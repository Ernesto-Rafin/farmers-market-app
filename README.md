# Farmers Market Pre-Order Application

This is a **Flask + SQLite web application** that allows users to browse farmers markets, view vendors and products, search for products, and place pre-orders that are stored in a database.

## Features

* Browse a list of farmers markets
* View market details, vendors, and products
* Search for products across all markets
* Place pre-orders using a database transaction
* Order confirmation after successful submission

## Technology Stack

* Python (Flask)
* SQLite
* HTML (Jinja templates)

## How to Run the Application

1. Make sure Python is installed
2. Initialize the database:

   ```
   python init_db.py
   ```
3. Start the Flask server:

   ```
   python app.py
   ```
4. Open a browser and go to:

   ```
   http://127.0.0.1:5000
   ```

## Project Purpose

This project demonstrates a **full life-cycle database application**, including data storage, retrieval, searching, and transactional pre-order processing.
